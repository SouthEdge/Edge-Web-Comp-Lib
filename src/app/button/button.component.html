Use the below custom attribute accordingly

Basic
  • btn - display the default butten with default title(submit) and border
  • btn-default - grey color background
Colors :
  • btn-primary - blue color background
  • btn-success - green color background
  • btn-warning - yellow color background
  • btn-danger - red color background
  • btn-info - blue color background
Size :
  • btn-wd - button width will be 140px
  • btn-fill - filled with dark grey color
  • btn-lg - large size button
  • btn-sm - small size button
  • btn-xs - very small size button
  • btn-round - round cornor button
  • btn-xs - very small size button
Styles :
  • btn-simple - simple button with border and color
  • btn-outline-default - grey color outline
  • btn-outline-primary - blue color outline
  • btn-outline-success - green color outline
  • btn-outline-warning - yellow color outline
  • btn-outline-danger- red color outline
  • btn-outline-info- blue color outline
